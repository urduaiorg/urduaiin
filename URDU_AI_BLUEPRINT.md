# اردو اے آئی: نمبر 1 اردو اے آئی ویب سائٹ بنانے کا بلیوپرنٹ

یہ دستاویز اس سوال کا عملی جواب ہے:

`اردو اے آئی کو ایسا کیسے بنایا جائے کہ یہ اردو بولنے والے لوگوں کی اے آئی سیکھنے کی پہلی پسند بن جائے؟`

---

## 1. اصل پوزیشننگ

اردو اے آئی کو ایک عام بلاگ، نیوز سائٹ، یا صرف ٹول ڈائریکٹری کے طور پر نہیں بنانا چاہیے۔

اس کی اصل پوزیشننگ یہ ہونی چاہیے:

**اردو اے آئی وہ جگہ ہے جہاں اردو بولنے والے لوگ آسانی سے اے آئی کو سمجھیں، سیکھیں، اور اپنے کام میں استعمال کریں۔**

یہ فرق بہت اہم ہے۔

اس کا مطلب:

- یہ سائٹ صرف خبروں کے لیے نہیں ہوگی
- یہ صرف definitions دینے کے لیے نہیں ہوگی
- یہ صرف “top 10 tools” قسم کا content فارم نہیں ہوگی
- یہ ایک `urdu-first learning product` ہوگی

---

## 2. اردو اے آئی کی جیت کس چیز میں ہونی چاہیے

اگر اردو اے آئی کو نمبر 1 بننا ہے تو اسے ان پانچ چیزوں میں سب سے بہتر ہونا ہوگا:

### 2.1. سب سے آسان سمجھ

قاری کو فوراً جواب ملنا چاہیے:

- یہ کیا ہے؟
- میرے کس کام آئے گا؟
- مفت ہے یا پیڈ؟
- موبائل پر چلتا ہے؟
- ابھی مجھے کیا کرنا چاہیے؟

### 2.2. اردو فرسٹ سوچ

اردو صرف زبان نہ ہو، پورا thinking model اردو فرسٹ ہو:

- headline اردو میں سوچا گیا ہو
- مثالیں مقامی ہوں
- tone دوستانہ ہو
- jargon فوراً explain ہو
- انگریزی صرف product names یا ضروری terms تک محدود ہو

### 2.3. بہترین reading experience

یہ سب سے underrated چیز ہے۔

زیادہ تر اردو websites:

- بھری ہوئی لگتی ہیں
- noisy ہوتی ہیں
- headings بہت aggressive ہوتی ہیں
- line spacing خراب ہوتا ہے
- mobile reading کمزور ہوتی ہے

اردو اے آئی کو اس کے برعکس ہونا چاہیے:

- calm
- airy
- focused
- mobile-first
- easy to scan

### 2.4. سچائی اور اعتماد

سائٹ کو ایمانداری سے بتانا چاہیے:

- کیا واقعی useful ہے
- کیا hype ہے
- کیا مفت ہے
- کہاں limitation ہے
- کہاں verify کرنا ضروری ہے

یہ honesty ہی brand trust بناتی ہے۔

### 2.5. habit loop

نمبر 1 site وہ نہیں جسے لوگ ایک بار دیکھیں۔

نمبر 1 site وہ ہے جس پر لوگ بار بار واپس آئیں۔

اس کے لیے ضروری ہے:

- start here path
- weekly updates
- AI news explained in Urdu
- tool reviews
- practical guides
- newsletter
- YouTube / WhatsApp loop

## 4. homepage کو کیسا ہونا چاہیے

homepage کو landing page نہیں، editorial front page کی طرح behave کرنا چاہیے۔

### homepage کے پہلے screen پر یہ 3 چیزیں فوراً نظر آنی چاہئیں:

1. اردو اے آئی کیا ہے
2. نئے قاری کو کہاں سے شروع کرنا چاہیے
3. آج کیا اہم/مفید چیز موجود ہے

### ideal homepage structure

1. compact hero
2. start here module
3. featured guide
4. top categories
5. latest tools / models
6. AI news in Urdu
7. newsletter module
8. footer with trust links

### homepage hero میں کیا ہونا چاہیے

headline:

`اردو میں اے آئی سیکھیں، آسان انداز میں`

subheading:

`گائیڈز، ٹولز، خبریں، اور عملی رہنمائی ایک ہی جگہ`

`یہاں سے شروع کریں`

secondary CTA:

`آج کی اہم گائیڈ دیکھیں`

hero کو poster نہیں بنانا۔ اسے gateway بنانا ہے۔

## 6. content pillars

اردو اے آئی کا content random نہیں ہونا چاہیے۔

اسے چار واضح pillars پر build ہونا چاہیے:

### pillar 1: explanation

ایسے صفحات جو concept سمجھائیں۔

مثال:

- اے آئی کیا ہے؟
- چیٹ جی پی ٹی کیا ہے؟
- Gemini کیا ہے؟

### pillar 2: decision help

ایسے صفحات جو قاری کو انتخاب میں مدد دیں۔

مثال:

- مفت اے آئی ٹولز
- ChatGPT vs Gemini
- Canva AI یا دوسرے design tools

### pillar 3: practical guides

ایسے صفحات جو قاری کو actual use تک لے جائیں۔

مثال:

- prompts کیسے لکھیں
- طلبہ کے لیے AI
- موبائل پر AI

### pillar 4: updates and trust

ایسے صفحات جو brand کو repeat visit اور authority دیں۔

مثال:

- ہفتہ وار AI نیوز
- tool updates
- privacy guidance

## 7. content quality rules

ہر article کے لیے یہ rules لازم ہونے چاہئیں:

### 7.1. article کا مقصد ایک ہو

ایک page ایک ہی main سوال solve کرے۔

### 7.2. پہلے 20 seconds میں clarity

ہر article کے شروع میں یہ ہونا چاہیے:

- یہ article کس چیز کے بارے میں ہے
- کس کے لیے ہے
- مختصر answer کیا ہے

### 7.3. Urdu dignity rule

تحریر translation جیسی نہیں لگنی چاہیے۔

غلط approach:

- direct English structure
- jargon heavy sentences
- half-Urdu half-tech noise

صحیح approach:

- سادہ Urdu
- natural rhythm
- short sections
- practical language

### 7.4. access checklist

ہر tool/model page میں یہ لازمی لکھا جائے:

- مفت یا پیڈ
- موبائل پر کام کرتا ہے یا نہیں
- login/region limitation
- beginners کے لیے ٹھیک ہے یا نہیں

### 7.5. editorial honesty

ہر tool کو بہترین ثابت نہ کریں۔

اگر کمزور ہے تو لکھیں:

- یہ beginners کے لیے نہیں
- یہ مہنگا ہے
- یہ mobile-first نہیں
- یہ overhyped ہے

## 8. design rules

### 8.1. visual direction

اردو اے آئی کا design feel ہونا چاہیے:

- calm
- intelligent
- modern
- warm
- editorial

نہیں ہونا چاہیے:

- cyberpunk
- generic startup
- over-animated
- cluttered news portal

### 8.2. typography

- Urdu headings کو breathing room دیں
- very large poster-scale headings avoid کریں
- line height generous ہو
- long paragraphs narrow reading column میں ہوں
- mixed Urdu/English lines کے لیے robust wrapping ہو

### 8.3. color system

base:

- white background
- deep green brand anchor
- warm gold accent

یہ combination trust + warmth دونوں دیتا ہے۔

### 8.4. component strategy

reuse ہونے والے components:

- category chips
- feature cards
- guide cards
- tool info blocks
- quick facts box
- newsletter module
- article TOC block

### 8.5. image strategy

ہر page کے لیے custom poster مت بنائیں۔

preferred system:

- text-led covers
- subtle SVG shapes
- clean icons
- screenshots جہاں واقعی helpful ہوں
- real photos صرف about / event / workshop contexts میں

## 11. publishing workflow

اردو اے آئی کے لیے publishing flow یہ ہونا چاہیے:

1. topic select کریں
2. user intent لکھیں
3. Urdu-first outline بنائیں
4. access checklist add کریں
5. publish سے پہلے clarity review کریں
6. internal links add کریں
7. social/newsletter loop سے connect کریں

### publish سے پہلے quality check

- کیا headline صاف ہے؟
- کیا intro فوراً بات سمجھاتا ہے؟
- کیا article useful ہے؟
- کیا Urdu natural لگ رہی ہے؟
- کیا page mobile پر اچھا لگے گا؟
- کیا reader کو next step مل رہا ہے؟
